El contenido de esta carpeta consiste en un conjunto de librerías que nos ofrecen diferentes métodos para aproximarse
a la raíz de una función, además cuenta con una estructura de datos que es instanciada en cada una de las librerías
la cual tiene como propósito almacenar secuencialmente cada aproximación, calcular el porcentaje de error de cada
aproximación apartir de la segunda iteración e informar si se encontró o no la aproximación con el porcentaje de error 
establecido.
